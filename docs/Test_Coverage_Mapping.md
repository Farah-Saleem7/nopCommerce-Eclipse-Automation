# Test Coverage Mapping

This automation project is based on the provided nopCommerce Test Cases Suite.

| Automation Class | Covered Area |
|---|---|
| AuthenticationTests | Register, validation, invalid login, logout |
| ProductTests | Search, product details, compare, add to cart |
| CartAndWishlistTests | Cart update/remove, wishlist add/remove |
| CheckoutTests | Guest checkout, Contact Us |
